I will download the file as CSV, which will be the input to the python script.

- the third line of the CSV contains the column names in English. read only up to the column name which is empty, ignoring all further columns
- the script will have two input parameters: the name of the CSV file and a party name
- given the input parameters, the script will create one JSON file per each row that has the same party name or "all"
- each output JSON file will be named the name of the party followed by a hyphen and then a 3-digit running index starting at 000.
- following is an example of the contents of one JSON:

  },
  "supporters": "unknown",
  "tags": [],
  "type": "official"

- the JSON "id" should be the same as the JSON file name (without the file extension)
- if the CSV party name is "all" the party name from the input parameter will be used
- if the CSV logo does not start with "http", and is not empty, attach to it the prefix "/inter-party-factions/faction_logos/"

- "supporters" and "tags" are for future use. keep them with the example data.
- links that are empty in the CSV should not be mentioned in the JSON
- note the use of Hebrew characters
